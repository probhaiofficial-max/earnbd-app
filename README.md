# EarnBD Super App — সম্পূর্ণ গাইড

## ফাইল কাঠামো
```
earnbd/
├── index.html          ← মূল অ্যাপ (এটি ডাবল-ক্লিক করে খুলুন)
├── css/
│   └── style.css       ← সব ডিজাইন
├── js/
│   ├── config.js       ← আপনার সেটিংস (শুধু এটি পরিবর্তন করুন)
│   ├── data.js         ← টাস্ক ও অফার তালিকা
│   └── app.js          ← সব লজিক
└── README.md           ← এই ফাইল
```

## শুরু করার ধাপ

### ১. লোকাল টেস্ট
- ZIP খুলুন
- `index.html` ডাবল-ক্লিক করুন — ব্রাউজারে অ্যাপ খুলবে

### ২. কনফিগার করুন (js/config.js)
```js
appName: "আপনার অ্যাপের নাম",
whatsapp: "+8801XXXXXXXXX",
telegram: "https://t.me/yourchannel",
```

### ৩. অ্যাড নেটওয়ার্ক যুক্ত করুন
অ্যাপের Settings পেজে গিয়ে ID বসান:
- **CPAGrip**: cpagrip.com → Publisher ID
- **CPALead**: cpalead.com → Website ID  
- **Adsterra**: adsterra.com → Social Bar কোড
- **Freecash**: freecash.com → রেফারেল লিঙ্ক

### ৪. ওয়ালেট সেটআপ
Settings-এ আপনার পেমেন্ট ঠিকানা বসান:
- bKash/Nagad নম্বর
- USDT TRC20 ওয়ালেট
- Bitcoin ঠিকানা

### ৫. Vercel-এ Deploy করুন (বিনামূল্যে)
1. github.com → New Repository
2. সব ফাইল আপলোড করুন
3. vercel.com → Import → Deploy
4. ২ মিনিটে লাইভ!
